Configuration Example

Here's how to configure the card in your Home Assistant Lovelace dashboard:

type: 'custom:meteofrance-weather-card' title: 'Next 24 Hours Forecast' location_id: '441430' # Your location ID

Features

  • Displays temperature and rain forecasts for the next 24 hours
  • Fetches data directly from Météo France API
  • Customizable location ID
  • Responsive design that works on different screen sizes

Installation

  1. Install via HACS (Home Assistant Community Store)
  2. Add the card to your Lovelace dashboard using the configuration example above
  3. Customize the location_id parameter to match your desired location
